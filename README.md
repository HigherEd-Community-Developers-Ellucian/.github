# Ellucian Community Code Hub

Welcome! This GitHub organization is a **community space** for sharing source code related to Ellucian products.

## Purpose
- Share practical integrations, scripts, examples, and utilities.
- Learn from peers across institutions.
- Collaborate in the open on community-maintained solutions.

## Important Disclaimer
- This organization is **not an official Ellucian support channel**.
- Code in these repositories is provided by community contributors.
- **No warranty, support commitment, or liability** is provided by Ellucian or contributing institutions.
- Use all code at your own risk, and validate it in your environment before production use.

## Before You Use Any Repository
1. Review the repository `README` and license.
2. Check compatibility with your Ellucian product/version.
3. Run security and quality checks internally.
4. Test in a non-production environment first.

## How to Contribute
Community contributions are welcome.

### Contribution flow
1. Open an issue to propose the change (bug, enhancement, or documentation update).
2. Fork the repository and create a focused branch.
3. Submit a pull request with clear context, setup steps, and validation notes.
4. Respond to review feedback and keep discussion constructive.

### Contributor expectations
- Include clear setup and usage instructions.
- Document assumptions and prerequisites.
- Add or update license information.
- Keep changes scoped and well explained.
- Be respectful and constructive in issues and pull requests.

## Becoming a Member
If you want to become an organization member or request collaborator access, submit the **Membership / Invite Request** form:

- [Open membership invite request](../../issues/new?template=membership-invite.yml)

Please include your GitHub username, institution, and how you plan to contribute so maintainers can review requests quickly.

## Support Expectations
- Maintainers and contributors may help when available, but support is best-effort.
- For official product support, use Ellucian’s official support channels.
